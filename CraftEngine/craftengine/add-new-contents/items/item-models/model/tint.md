# 🎨 染色

所有能使用颜色的地方都支持十进制或 RGB 格式

```yaml
value: 16711680
```

```yaml
value:
  - 255
  - 0
  - 0
# or
value:
  - 1.0
  - 0.0
  - 0.0
```

```yaml
value: 255,0,0
# or
value: 1.0,0.0,0.0
```
